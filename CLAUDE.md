# UnitreeG1ハッカソン

- Unitree G1をWebから動かしたい！リポジトリ

## アイデア

- ロボットと人間のAIエージェントに適したUI/UXを作りたい．
- 生活環境で，介護や生活支援の文脈で，家事労働を代行してくれるUnitree G1．作業の上達の様子を見せる．メンタルケア，子供のお留守番の相手になる，高齢者のサポート，家事代行．
- 家の中の特定のエリア（机の上とか）に行ってほしいとUI上で操作すると，
- お片付け支援，G1が片づけるべき場所を教えてくれる．
- 子供がG１に質問すると，インタラクティブな会話形式で，実際に案内してくれる．
- 高齢者がどこに封筒を置いたっけ？とか思ったときに，教えてれる

## 技術スタックの想定

- Web UI（フロントエンド）：Vue.js
  - bun
- バックエンド：FastAPI(Python)
  - uv

## WebUIのイメージ

- MVP
  - キッチン、テーブル、トイレのラベルがある
  - ラベルを押すと、ロボットが移動するかどうか確認ダイアログ
  - ボタンが押されたら、PythonのUnitree G1用の関数を動かすプログラム

- フェーズ2
  - 音声入力
  - マイクの許可
  - キッチン行って行ったらその通り移動する

## Unitree G1の技術イメージ

- ROS2 を用いて SLAM + Navigation を実装
- web インターフェースからいきたいロケーションを選択（キッチン，テーブルなど）
- 移動中は障害物回避をする（アピールポイント？？）
